关于GitHub Pages绑定独立域名

时间: 2015-09-13 00:21:53 标签: gh-pages · github · 域名

本文目标

  • 将本站startexample.com域名绑定到GitHub gh-pages分支上。 注:本篇博文主要介绍如何将顶级域名解析到GitHub gh-pages repo上!

说明

本站绑定的为顶级域名,更多域名绑定方式可见更多域名绑定方式。

前提

在GitHub的repo上有博客的gh-pages分支且已购买域名startexample.com。

步骤

首先,在gh-pages的repo根目录中新建不带后缀的CNAME文件,文件中写上域名如startexample.com域名不带前缀www。

其次到域名提供商控制面板管理界面,添加两个A记录到192.30.252.153以及192.30.252.154,注意这里的ip地址随时可能更改,需要在GitHub域名解析处确认解析ip地址。
