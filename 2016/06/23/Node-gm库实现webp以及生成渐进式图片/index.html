Node gm库实现webp以及生成渐进式图片

时间: 2016-06-23 16:00:19 标签: gm · node · webp · 渐进式图片

提示: 本文基于Centos 7.

安装GraphicsMagick

Centos中使用如下命令安装, 默认支持webp格式:

$ yum install GraphicsMagick

使用如下命令查看是否安装成功以及支持webp格式:

$ gm -version

渐进式图片生成

目前GraphicsMagick可将(PNG, GIF, JPEG)三种格式图片转换成渐进式图片格式.

  • 命令行生成渐进式图片

可使用如下CLI命令生成渐进式图片: $ gm convert 3.jpg -interlace Line 3-line.jpg
